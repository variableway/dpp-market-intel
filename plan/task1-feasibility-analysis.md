# Task 1: DPP 数据采集 — 可行性分析

## 1. 需求概述

Task 1 需要采集以下三类数据：

| # | 数据需求 | 具体内容 |
|---|---------|---------|
| A | DPP 申报数据的分品类数量统计 | 19 个品类的出口额、计费单元、目标企业数等 |
| B | 2024/2025 出口额、计费单元、目标企业数 | 中国对欧盟出口数据 |
| C | 海关公开数量可用度和待补拉 HS 查询清单 | 分品类的海关数据覆盖率和缺口 |

## 3. 数据来源可行性分析

### 3.1 UN Comtrade API（推荐 — 主要数据源）

| 项目 | 详情 |
|------|------|
| **数据覆盖** | 全球双边贸易数据，按 HS 编码分类（2/4/6 位） |
| **中国数据** | 中国（reporter code `156`）按 HS 章节报告出口数据 |
| **欧盟粒度** | 需逐一查询 27 个欧盟成员国，无 EU27 聚合代码 |
| **API 端点** | `https://comtradeapi.un.org/data/v1/comtrade/final/get` |
| **认证** | 免费注册获取 `Ocp-Apim-Subscription-Key` |
| **免费额度** | 500 次/天，每次 10 万条记录 |
| **数据滞后** | 年度数据滞后 12-18 个月；2024 数据部分可用，2025 尚不可用 |
| **Python 库** | `pip install comtradeapicall` |

**可行性：高。** 免费额度完全满足需求（27 国 × 按章查询 = 约 54 次/年）。

**覆盖缺口：**
- 2025 年年度数据预计 2026 年中后期才可获取
- 部分精细 HS 子目可能需要更多 API 调用
- 海关数量（件/吨）的可用性取决于 HS 编码的报告质量

### 3.2 Trading Economics

| 项目 | 详情 |
|------|------|
| **数据覆盖** | 按国家/商品大类的宏观贸易统计 |
| **优势** | 数据更新较快，覆盖 2024-2025 |
| **API** | 付费 API（$249/月起），免费版有限 |
| **当前用途** | 项目现有 CSV 数据的主要来源 |

**可行性：中。** 当前数据已依赖此来源。但付费 API 成本较高，建议仅作为辅助验证来源。

### 3.3 中国海关统计数据查询平台 (stats.customs.gov.cn)

| 项目 | 详情 |
|------|------|
| **数据覆盖** | 中国海关原始统计，含 HS 8 位编码 |
| **访问方式** | 网页交互查询，无公开 API |
| **反爬措施** | 滑块验证码 (CAPTCHA) |
| **导出限制** | 每次查询最多 10,000 条记录 |
| **数据粒度** | 比 UN Comtrade 更细（HS 8 位中国国家编码） |

**可行性：中低（自动化角度）。** 无 API，需要手动查询或复杂爬虫方案。但数据质量最高，适合作为校验基准。

**推荐用途：** 定期手动查询关键品类的 EU 出口数据，特别是 UN Comtrade 覆盖不足的品类（如家具件数、玩具件数等）。

### 3.4 EU Eurostat Comext 数据库

| 项目 | 详情 |
|------|------|
| **数据覆盖** | 欧盟从中国进口的镜像数据 |
| **API** | Eurostat REST API，免费 |
| **优势** | 可作为 UN Comtrade 的交叉验证来源 |

**可行性：中。** 可作为辅助验证，但不建议作为主要数据源。

### 3.5 EU DPP/ESPR 法规数据

| 项目 | 详情 |
|------|------|
| **ESPR 生效** | 2024 年 7 月 18 日 |
| **首批工作计划** | 2025 年 4 月发布（确定优先品类） |
| **首批授权法案** | 预计 2026 年 2 月 |
| **首批 DPP 强制** | 2027 年（电池、纺织等优先品类） |
| **来源** | EUR-Lex, 欧盟委员会官网 |

**可行性：高。** 法规数据为公开信息，可通过 EUR-Lex API 或网页获取。但 DPP 品类清单仍在制定中，需持续跟踪。

## 5. 可行性总结

### 5.1 整体可行性评估

| 维度 | 评级 | 说明 |
|------|------|------|
| 出口金额数据 | **高** | UN Comtrade 可覆盖所有品类，免费可用 |
| 出口数量数据（件/吨/条） | **中** | 部分品类有（手机、汽车、铝材），部分缺失（纺织、家具、玩具） |
| 目标企业数 | **低** | 无公开数据源，需基于行业报告或估算 |
| 海关数据可用度 | **高** | 已有清单，可定期更新 |
| HS 查询补拉 | **中** | UN Comtrade + 海关交互查询可覆盖大部分 |
| 法规时间线 | **高** | EUR-Lex 公开可获取 |
| 数据自动化 | **中** | UN Comtrade API 可自动化，海关需手动 |

### 5.2 推荐方案

1. **主要数据源：UN Comtrade API** — 自动化采集所有品类对欧盟出口金额
2. **辅助数据源：海关统计数据查询平台** — 手动查询缺失的数量数据
3. **交叉验证：Trading Economics / Eurostat** — 确认数据一致性
4. **法规跟踪：EUR-Lex / EC 官网** — 跟踪 DPP/ESPR 品类和时间线更新
5. **企业数估算：行业报告 + 比例推算** — 维持当前估算方法论

### 5.3 风险和限制

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 2025 年贸易数据尚不可用 | 无法获取最新数据 | 使用已有 Trading Economics 数据 + 2024 基线推算 |
| 海关数量数据不完整 | 部分品类无法验证计费单元 | 维护查询清单，定期手动补拉 |
| 无公开企业级数据 | 企业数仅能估算 | 标注数据来源和估算方法 |
| EU27 无单一聚合代码 | API 调用次数增加 | 27 国循环查询 + 聚合脚本 |
| DPP 品类清单未最终确定 | 时间线可能变化 | 持续跟踪 ESPR 工作计划更新 |
